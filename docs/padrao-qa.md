# PADRÃO QA — CRITÉRIOS E PROMPT

## REGRAS OBRIGATÓRIAS

* Usar os 8 blocos: F, A, I, U, R, B, N, RG
* Cada bloco deve ter no mínimo 1 critério ou N/A
* Máximo recomendado: 3 critérios por bloco
* Cada critério deve ser verificável (verdadeiro/falso)

## PROIBIDO

* Explicações longas
* Critérios genéricos
* Repetição entre blocos

## OBJETIVO

Critérios devem ser:

* Diretos
* Testáveis
* Sem ambiguidade
